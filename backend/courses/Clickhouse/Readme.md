## What is ClickHouse

- ClickHouse = Clickstream data WareHouse
- Database Management System
- Open-source
- OLAP (Online Analytical Processing)

## Why ClickHouse

- ClickHouse uses columnar storage
    - A column's values are stored in a file with nothing but it's values
    - Compression algorithms ar more efficient and greatly compress the data
- Scale: 1 billion rows/sec
- ClickHouse is a SQL database management system
    - Query your data without having to learn a new query language
    - SQL is flexible to other languages (e.g. Datadog)
